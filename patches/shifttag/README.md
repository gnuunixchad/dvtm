### Description

Shift to next/previous tag, with skipping occupied/unoccupied variants.

| function | behavior |
| :--- | :---|
| shifttag  | shift to next/previous tags |
| shifttag_occupied  | skipping unoccupied tags |
| shifttag_unoccupied | skipping occupied tags |
